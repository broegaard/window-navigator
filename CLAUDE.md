# CLAUDE.md

Keyboard-driven window switcher for Windows. Double-tap **Ctrl** to open an overlay listing all open windows; type to filter, arrow keys to navigate, Enter to focus.

## Repository layout

| Directory | Contents |
|-----------|----------|
| `python/` | Original Python implementation |
| `go/` | Go port (complete — the shipping binary) |

Each subdirectory has its own `CLAUDE.md` with commands, architecture, and gotchas.
